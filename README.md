**BMI Calculator**

A simple BMI (Body Mass Index) Calculator built using React and Vite.  
The app calculates BMI and shows the result on a separate page using React Router.

**Features**
- User inputs height and weight
- Calculates BMI instantly
- Displays result on a separate page
- Clean and responsive UI
- React Router navigation

**Project Structure**
```

src/
├── pages/
│    ├── BMICalculator.jsx
│    ├── BMICalculator.css
│    ├── Result.jsx
│    ├── Result.css
├── App.jsx
├── main.jsx

````
**BMI Formula**

BMI = weight (kg) / (height (m) × height (m))
